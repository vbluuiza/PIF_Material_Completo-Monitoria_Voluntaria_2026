# Projeto de Apoio à Disciplina de PIF

Este repositório foi criado para auxiliar estudantes da disciplina de PIF da Cesar School. Aqui você encontrará uma coletânea de exercícios, exemplos e materiais organizados por temas, todos desenvolvidos e reunidos durante minha experiência como aluna e monitora voluntária da discplina.

## Estrutura do Projeto

O projeto está organizado em pastas temáticas, cada uma abordando um tópico fundamental da disciplina:

- **Exercícios C para AV2**: Exercícios diversos para revisão e prática para AV2.
- **Lista de Exercícios 01 - Estrutura de Controle**: Exercícios sobre estruturas de controle (if, else, switch, laços).
- **Lista de Exercícios 02 - Vetores e Matrizes**: Prática com arrays e matrizes em C.
- **Lista de Exercícios 03 - Funções**: Exercícios focados em modularização, funções e procedimentos.
- **Lista de Exercícios 04 - Strings**: Manipulação de strings em C.
- **Lista de Exercícios 05 - Tipos Compostos**: Estruturas, uniões e enums.
- **Lista de Revisão**: Exercícios para revisão geral.
- **Manipulação de Arquivos**: Exemplos e exercícios de leitura e escrita em arquivos.
- **PIF**: Questões variadas e exemplos práticos.
- **Ponteiros**: Exercícios e exemplos sobre ponteiros.
- **String**: Exemplos e exercícios de manipulação de strings.
- **Struct, union**: Exemplos de uso de struct e union.
- **Tipos Compostos**: Exemplos de enum, struct e union.
